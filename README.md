# Music Player

一个支持本地音频播放和网易云音乐在线搜索的网页音乐播放器。

## 功能

- **本地播放**：拖拽或选择 MP3、FLAC、AAC、OGG、WAV 文件
- **在线搜索**：通过网易云音乐 API 搜索并播放歌曲
- **播放列表**：增删、拖拽排序、右键菜单操作
- **进度控制**：点击/拖拽进度条跳转
- **音量控制**：滑动调节、一键静音
- **播放模式**：顺序 / 列表循环 / 单曲循环 / 随机播放
- **歌词显示**：同步滚动歌词，点击跳转
- **频谱可视化**：圆形频谱动画
- **快捷键**：空格播放暂停、方向键进度/音量、N/P 下/上一首、M 静音

## 运行方法

### 1. 安装依赖

```bash
npm install
```

### 2. 启动服务

```bash
npm start
```

### 3. 打开浏览器

访问 [http://localhost:3000](http://localhost:3000)

## 依赖说明

| 包名 | 用途 |
|------|------|
| `express` | HTTP 服务器 |
| `cors` | 跨域支持 |
| `NeteaseCloudMusicApi` | 网易云音乐 API |

> **注意**：在线搜索功能需要服务端正常运行。如果只使用本地播放，也可以直接用浏览器打开 `index.html`（不支持在线功能）。
